## Why

背雅思单词时，传统的"看单词→背释义"方式过于被动，拼写能力常常被忽视。很多人认识一个词但拼不出来，或者在写作中拼错。本工具提供一种主动、有时限、有反馈的拼写训练方式——给你中文释义，你在 15 秒内敲出英文单词，实时纠错，用类似"听写"的体验强化拼写记忆。

## What Changes

- 新增一个纯前端、零后端的单页应用——"雅思拼写训练器"
- 用户打开网页即可开始拼写训练，无需登录
- 展示中文释义，15 秒倒计时内输入对应英文单词
- 自动播放真人发音（通过 dictionaryapi.dev 免费 API 获取 MP3）
- 输入实时检测，敲错立即整框标红
- 超时后冻结界面，红色显示正确单词
- 一轮 15 个单词，结束后展示统计（正确数/错误数/成功率）
- 支持复习错词模式，错词记录持久化到 localStorage
- 内置 500 精选雅思高频词库（中文释义本地维护）
- 支持导出错词列表

## Capabilities

### New Capabilities
- `spelling-game`: 核心拼写游戏流程——显示中文、倒计时、实时输入检测、发音播放、超时/错误处理
- `word-round`: 轮次管理——15 词一轮、随机抽词、不重复、轮次切换
- `wrong-words`: 错词记录与管理——localStorage 持久化、复习模式、导出功能
- `word-bank`: 词库——本地 JSON 词库加载、API 发音/音标/例句增强、降级策略

### Modified Capabilities

无（新项目，不修改现有能力）

## Impact

- 新增静态文件：`index.html`, `style.css`, `app.js`, `data/words.json`
- 无后端依赖，无数据库
- 依赖外部 API：dictionaryapi.dev（发音 MP3、音标、例句）——纯前端调用，不需要 API Key
- API 不可用时的降级逻辑——核心拼写功能不依赖外部 API
